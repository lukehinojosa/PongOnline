## Run Instructions

1. Download [Signaling-WindowsBuild.zip](https://github.com/lukehinojosa/PongOnline/releases), then unzip it and run pong_signaling.exe. (Only 1 person has to do this).
2. Copy the link provided by the pong signaling console.
3. All players go to [Pong Online](https://lukarbonite.itch.io/pong-online), then paste the signaling link into the signaling text box.
4. This supports multiple parties, but parties are up to 2 players.
5. 1 player chooses to be the host, then gives the other player the code to join.
